Lista de teclas:

Cambiar modelos
    Flecha Derecha - Modelo Siguiente
    Left Arrow - Modelo Anterior
    Arrastrar Mouse - Rotar Modelo
    Rueda Mouse - Zoom Modelo

Shaders:
    r - Reiniciar Shaders
    1 - Gourad  Shader
    2 - Blanco y Negro Shader
    3 - Pixeladot Shader
    4 - Iluminacion Shader
    5 - RGB Shader

Camara:
    w - Subir
    s - Bajar
    a - Rotar Izquierda
    d - Rotar Derecha

SpaceBar - Pausar o Reanudar Musica
